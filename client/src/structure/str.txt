## Quick setup commands

    ```bash
# 1. Install deps
npm install

# 2. Add remaining shadcn components not in stubs
npx shadcn@latest add button input label

# 3. Copy .env.example → .env.local and fill in values
cp .env.example .env.local

# 4. Start dev server
npm run dev
```
